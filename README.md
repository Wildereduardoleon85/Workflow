# Workflow

Aplicación demo donde se pueden ingresar varios Nros. de identificación en la barra de búsqueda y muestra un resultado de avance ("workflow")
se adjunta un dump de la base de datos hecha con MySQL, para poder utilizar se debe ejecutar en un servidor local tal como Xampp (Apache). 
Realizado con PHP para el Backend y CSS y JQuery para el FrontEnd.

English: 
Demo application where you can enter several Nos. identification in the search bar and shows a progress result ("workflow")
A dump of the database made with MySQL is attached, in order to use it it must be run on a local server such as Xampp (Apache).
Made with PHP for the Backend and CSS and JQuery for the FrontEnd.
